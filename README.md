# Exercícios de Lógica com JavaScript

Este repositório reúne os exercícios de lógica que estou desenvolvendo durante meus estudos em JavaScript.

A proposta é registrar minha evolução de forma organizada, começando por fundamentos como variáveis, operadores e condicionais, e avançando progressivamente para funções, objetos, arrays, métodos e problemas mais completos de lógica de programação.

## Objetivos deste repositório

- Praticar lógica de programação
- Fortalecer minha base em JavaScript
- Registrar minha evolução com exercícios progressivos
- Criar um histórico de aprendizado visível no GitHub
- Servir como base para desafios mais complexos no futuro

## Progressão de estudos

Os exercícios seguem uma evolução gradual, incluindo temas como:

- tipos de variáveis
- operadores
- condicionais
- funções
- validações
- regras de negócio
- manipulação de dados

Futuramente, o repositório também deverá incluir:

- objetos
- arrays e métodos
- manipulação de formulários
- integração entre HTML, CSS e JavaScript
- exercícios mais próximos de cenários reais

## Estrutura

Cada pasta representa um exercício independente e contém:

- `index.js`: código principal do exercício
- `README.md`: descrição do problema, regras, entradas, saídas e conceitos praticados

## Observação

Este repositório representa meu processo de evolução. Por isso, os exercícios mais antigos podem ter soluções mais simples, enquanto os mais recentes tendem a apresentar regras, validações e estruturas mais completas.
